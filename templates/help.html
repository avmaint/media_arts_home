Digital Tools Overview

A guide to the software systems used by the UAC Media Arts team

The UAC Media Arts team uses several interconnected web-based tools. This page describes what each one does, who uses it, and when. All tools are accessible from the Media Arts home page and are intended for internal use only.
🏠
Media Arts Home
port 80  ·  this site

The central portal for the Media Arts team. It provides quick access to every tool, device, and resource the team uses — organised into Operational, Administration, and Reference sections.

It also displays time-limited announcement banners for the team (configured under Administration → Message Admin).

  • Links to all operational tools, consoles, and device web interfaces
  • Links to system documentation and equipment manuals
  • AV glossary (pulled live from the AVL Data database)
  • Team announcement banners with priority levels

CueCommander
cumu-g001.local:1880/dashboard

The AV automation and cue control dashboard, built on Node-RED. Used primarily by the technical director and audio operator during services and events.

The Assignments page is used before each service to set microphone names, vocalist names, and IEM pack assignments — then push those names to all connected systems (dLive console, Klang monitors, Shure Wireless Workbench, Reaper) in one step.

  • Mic and pack assignments — set once, push to all systems
  • Lighting cue control (ETC ColorSource)
  • PTZ camera preset recall
  • Projector on/off and input switching
  • Klang personal monitor mix management

Companion
cumu-g001.local:8000

Companion provides a grid of programmable buttons, accessible from any browser or from a physical Stream Deck. It is used for a few key functions.

  1. We use a Stream Deck located at the video desk for controlling the ATEM video switcher.
  2. We use a Stream Deck located at the ProPresenter deck for manafing the power on/off of projectors and TVs.
  3. We have a Stream Deck located in the Nursery area used for calling a parent by number.
  4. Other features maybe added over time.
  • One-tap triggering of CueCommander actions
  • ProPresenter slide control
  • Lighting and video routing shortcuts
  • Configurable button layouts per operator role

AVL Tech Assistant
cumu-g001.local:8080

An AI-assisted knowledge and asset tool for Media Arts technical staff. It lets you search the equipment inventory, view device details and manuals, browse the AV knowledge base, and look up glossary terms — all in one place.

Useful for troubleshooting, training, and researching unfamiliar equipment.

  • Full asset inventory with tags, locations, and specs
  • Links to equipment manuals (also browsable under Reference → Equipment Manuals)
  • Knowledge base articles for common issues and procedures
  • AV glossary (the source that feeds the Media Arts glossary page)
  • Network device status monitoring
Open AVL Tech Assistant ↗
🗄️
AVL Data
cumu-g001.local:8001

The administrative database interface for structured AVL data. This is where asset records, glossary entries, and other reference data are created, edited, and maintained.

Most team members will interact with this data indirectly through the AVL Tech Assistant. AVL Data is primarily used by administrators who need to add or correct records directly.

  • Create and edit asset records (equipment inventory)
  • Manage glossary terms and definitions
  • Direct table-level access to all structured AVL data
  • Changes here appear immediately in AVL Tech Assistant and the Glossary
